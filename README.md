# Speed Test

Инструмент для замера скорости интернета. Последовательно выполняет N HTTP-запросов к указанному URL (по умолчанию — тяжёлая картинка), измеряет среднее время ответа и выводит скорость загрузки в МБ/с.

## Требования

- Python 3.12+
- [Poetry](https://python-poetry.org/)

## Установка и запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/sashabaranov/speed-test.git
cd speed-test

# Установить зависимости
make install

# Запустить с параметрами по умолчанию (10 запросов к изображению Wikimedia)
make run

# Или напрямую с кастомными параметрами
poetry run python main.py --url https://example.com/image.jpg --count 5
```

## Параметры CLI


| Параметр  | Описание                                       | По умолчанию                        |
| --------- | ---------------------------------------------- | ----------------------------------- |
| `--url`   | URL ресурса для замера (тяжёлый файл/картинка) | Постер с котиками Wikimedia (~6 МБ) |
| `--count` | Количество последовательных запросов           | 10                                  |


## Пример вывода

```
==================================================
       РЕЗУЛЬТАТЫ ЗАМЕРА СКОРОСТИ
==================================================
  Количество запросов : 10
  Скачано данных      : 61.45 МБ
  Среднее время запр. : 0.523 сек
  Скорость            : 11.74 МБ/с
==================================================
```

## Команды Makefile


| Команда             | Описание                                   |
| ------------------- | ------------------------------------------ |
| `make install`      | Установить все зависимости через Poetry    |
| `make run`          | Запустить замер с параметрами по умолчанию |
| `make test`         | Запустить тесты (pytest)                   |
| `make lint`         | Проверить стиль кода (black, isort, mypy)  |
| `make format`       | Автоформатировать код (black + isort)      |
| `make docker-build` | Собрать Docker-образ                       |
| `make docker-run`   | Запустить замер в Docker-контейнере        |


## Запуск в Docker

```bash
# Собрать образ
make docker-build

# Запустить с параметрами по умолчанию
make docker-run

# Запустить с кастомным URL
docker run --rm speed-test --url https://example.com/bigfile.jpg --count 5
```
